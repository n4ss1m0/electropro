Multimetro Laser









Este proyecto integra diversos sensores para realizar múltiples mediciones:

Medición de Distancia: Utiliza sensores de luz infrarroja para medir la distancia.
Medición de Longitud: Emplea un codificador rotatorio y una rueda impresa en 3D.
Medición de Nivel y Ángulos: Utiliza un módulo IMU (Unidad de Medición Inercial) para medir tanto el nivel como los ángulos.
Medición de RPM: Utiliza otro tipo de sensor infrarrojo.

El dispositivo también incluye una batería y un módulo de carga con conexión USB. Para controlar los distintos modos y funciones, cuenta con tres botones. Los valores medidos se muestran en una pantalla OLED.
